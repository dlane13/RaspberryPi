About The Project

Using Raspberry Pi to Optimize 3D Printing

Team CD: Darby Lane and Callie Levitt

3D printers have revolutionized the maker movement throughout the world, and this project aims to enhance the accessibility of printers for makers who need remote access to efficiently produce projects. Using a Raspberry Pi, our team has developed a remote monitoring system for 3D printing enthusiasts to track their projects and prototype a conveyor belt. If implemented, this system allows for continuous 3D printing workflows without the need for physical presence in the lab.
