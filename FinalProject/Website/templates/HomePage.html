About The Project

Using Raspberry Pi to optimize 3d Printing

Team CD : Darby Lane and Callie Levitt

3D printers have revolutionized the maker movement throughout the world, and this project will revolutionize the accessibility of printers to makers who need remote access in order to efficiently produce projects. Through the use of a Raspberry Pi, our team has worked to create a remote monitoring system for 3d printing enthusiasts to keep track of their projects, as well as prototyping a conveyor belt, which if implemented within the printer, would allow for a continuous flow of 3d prints without ever needing to set foot in the lab.
